Git
Apa itu Git ?

Git adalah salah satu website yang berguna untuk menyimpan source code kita dan mencatat perubahan code yang kita ubah

Manfaat apa yang kita dapat dari git?

•Mempermudah kerja sama orang lain dalam jarak jauh.

•Bisa mencari/melihat code opensource.

•Memudahkan mencari code apa saja yang diganti dikarnakan ada log.

Kapankah git dibutuhkan ?

•Saat ingin membuat aplikasi/code agar bisa di back-up.

•Membuat perubahan disuatu code.

•Ingin kerja/kolaborasi bersama orang lain.
